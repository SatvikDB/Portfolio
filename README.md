# Satvik DB - Personal Portfolio

A modern, responsive personal portfolio website showcasing my projects, skills, and experience as a Software Engineering student specializing in AI and Full-Stack Development.

## Tech Stack

- React 18
- Vite
- CSS3 (Custom styling with modern design)
- Responsive Design

## Features

- Modern dark theme with green accents
- Smooth scroll navigation
- Responsive mobile-first design
- Featured projects showcase
- Skills categorization
- Certifications display
- Contact information with social links

## Installation & Setup

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Sections

- **Hero** - Introduction and call-to-action
- **About** - Professional summary and education
- **Skills** - Technical skills organized by category
- **Projects** - Featured projects (AEGIS, ClassVisionAI, KrishiRaksha)
- **Certifications** - AWS and Google Cloud certifications
- **Contact** - Get in touch section with social links
